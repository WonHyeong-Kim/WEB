함수의 표현식

함수를 선언하고 사용할 수 있지만 표현식으로 정의하고 사용하는 것이 권고사항 표현시으로 함수를 정의하면 함수의 호이스팅이 일어나지 않는다.
[형식] - 이름이 있는 함수의 표현식 var 변수명 = function 함수명(매개변수...)
// 실행할 코드
}
=연산자를 이용해서 변수에 함수를 할당하는 표현 정의된 함수가 할당된 변수가 마치 함수의 이름인 것 처럼 쓰여짐. 따라서 원래 함수의 이름은 내부에서만 사용되고 함수외부에서 호출하는 경우는 함수가 할단된 변수명을 함수명으로 사용한다.


[형식] - 이름이 없는 함수의 표현식 - 익명함수
var 변수명 = function (매개변수...)
// 실행할 코드
}
